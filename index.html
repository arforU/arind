Tentang Web Ini

Tujuan dibuatnya website ini adalah untuk memberikan wawasan dan pemahaman kepada siswa dan pengguna umum tentang layanan web hosting, jenis-jenisnya, serta prinsip kerjanya secara sederhana dan mudah dipahami.

A. Pengertian Share Hosting Server

Layanan shared hosting sangat dipengaruhi oleh perilaku dari semua penyewa jasa hosting yang bersangkutan. Ciri khas shared hosting adalah harga yang murah dengan spesifikasi server yang cukup baik untuk kebutuhan dasar.

1. Shared Hosting

Shared hosting adalah layanan hosting dimana sebuah server digunakan secara bersama-sama oleh banyak pengguna sekaligus. Para pengguna tersebut berbagi resource server yang sama dan memiliki hak akses terbatas.

B. Prinsip Kerja Share Hosting Server

Server shared hosting digunakan bersama oleh banyak pengguna dan berjalan pada sistem operasi dan control panel yang sama. Masing-masing pengguna tidak mengetahui siapa saja pengguna lainnya dan berbagi resource secara terbatas.

C. Jenis-jenis Layanan Hosting

  1. Shared Hosting: Satu server digunakan bersama-sama oleh banyak pengguna Shared hosting adalah layanan hosting dimana sebuah server digunakan secara bersama-sama oleh banyak pengguna sekaligus. Para pengguna layanan shared hosting tersebut berbagi resource & kapasitas server bersama-sama. User memiliki hak akses terbatas ke server dan setting server sudah ditentukan oleh pengelola server..
  2. VPS Hosting: Server fisik dibagi menjadi beberapa server virtual Pengguna memiliki root akses VPS server masing-masing dan memiliki wewenang penuh melakukan pengaturan server VPS-nya. Untuk mengelola server VPS, Anda membutuhkan pengetahuan dasar mengenai server..
  3. Cloud Hosting: Hosting dengan banyak server terhubung agar performa tetap stabil Web hosting jenis cloud merupakan paket yang paling banyak disukai di pasaran karena lietrally tanpa downtime. Ciri khas dari web hosting jenis Cloud yaitu memilki banyak server yang didesain untuk berinteraksi dan terhubung satu sama lain agar performa situs web tetap terjaga. Dengan memakai hosting jenis ini tidak ada kekhawatiran terkait keterbatasan sumber daya maupun ketidakstabilan server..
  4. Dedicated & Colocation: Server eksklusif, baik disewa maupun milik pribadi yang diletakkan di data center.
  5. Wordpress Hosting: Hosting khusus untuk pengguna WordPress, dilengkapi fitur tambahan Disamping itu paket web hosting jenis ini akan menyertakan fitur-fitur tambahan..
